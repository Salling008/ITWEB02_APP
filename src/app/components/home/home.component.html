Wolcome to Workout Buddy.

In this website you can create your own workouts with self-selected exercises.

In Workouts you can see all the workouts users can do

To create your own Workout, edit a workout or log the workouts you have completed, you must have created a user and be logged in.

You can use the Register page to create your user and Login to log in on it.

Once logged in, you can create your Workout using Create Wourkout.
